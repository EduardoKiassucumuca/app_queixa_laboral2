# Plataforma Para Gestão de Queixas Laborais
Plataforma para gestão de queixas laborais

Requirements

Este projecto está a ser feito usando as seqguintes tecnologias:

ReactJS, NodeJS e o MySQL.

Como rodar a aplicação

Client:
1. $ cd client
2. $ npm install
3. $ npm start

Server:
1. $ cd server
2. $ yarn dev
